# Emlak Haritası + Chatbot

Bu proje Flask backend + HTML/CSS/JS frontend ile çalışır.

## Gerekli dosyalar

Aynı klasörde şu dosyalar olmalı:

- index.html
- style.css
- map.js
- app.py
- requirements.txt
- listings.json
- district_centers.json

## Kurulum

Terminali proje klasöründe aç:

```bash
pip install -r requirements.txt
python app.py
```

Sonra tarayıcıda aç:

```text
http://127.0.0.1:5000
```
